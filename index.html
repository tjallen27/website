Stripe Subscription

This application was written in NodeJS and uses Stripe API to process payments and successfully adds customers to a subscription list which is linked to a mailing system. Barzurk

This is a V2 of a visual job searching app I created at General Assembly built with NodeJS, ExpressJS and Mongo. It allows bars/pubs/restaurants to register and submit jobs which can then be viewed on a map for job seekers.

Solace Property Rental

Solace was created to skip out the middle man in propert rental, allowing people to email the landlord directly through the app, avoiding agency fees. The app was built with the intention of improving on Rails development.
